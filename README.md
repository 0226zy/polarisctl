# Polarisclt

> The polaris-server command line tool is used to quickly initiate related OpenAPI requests

## Quickstart

* Help

 ```go
 go run main.go -h
 ```

* get namespace list

```go
 go run main.go namespaces list --print=name,owners,comment -l10
```
